# Zepto-sales-sql-analysis
SQL analysis project using Kaggle Zepto dataset

## Project Overview
This project analyzes sales data from a Zepto dataset to answer real business questions using SQL.

## Tools Used
- MySQL
- Kaggle Dataset
- SQL

## Key Analysis Performed
- Data Cleaning Of NULL MRP Products
- Top-selling products
- Stock Details
- Category-wise Estimated revenue
- Top 10 best value prodcuts based on discount

## Skills Demonstrated
- SQL queries
- GROUP BY & HAVING
- Subqueries
- Window Functions
- Business insights
